Portfolio Description for Website Sistem Informasi Rumah Sakit Telang Sehat

1. System Overview

The Hospital Information System is a personal web-based project developed to support digital hospital administration. This application manages data related to doctors, nurses, guardians, patient profiles, and hospital rooms. The system is built using the Laravel framework, which provides modern structure, security, and scalability.

The main objectives of this project include:

  • Digitizing patient and medical staff records
  • Managing treatment records efficiently
  • Providing user-based access control
  • Improving data integrity and hospital workflow documentation

3. System Planning Process

a. Initial Problems Identified
  • Manual patient data recording
  • No centralized database for medical records
  • Difficulty tracking treatment changes
  • Room allocation not well documented
b. Planned Solutions
  • Build structured patient data management
  • Provide digital access to doctors and nurses
  • Store treatment progress safely in a database
  • Develop features for communication and notification handling

4. System Designing

a. Database Structure

The database consists of several important tables:

  • users: Stores login credentials for admin, nurse, doctor
  • patients: Contains patient biodata, treatment notes, medical status
  • doctors: Stores doctor profiles and specialization
  • nurses: Contains nurse information
  • guardians: Stores guardian data of each patient
  • rooms: Contains room names, types, and patient allocation
b. Features Designed
  • CRUD (Create, Read, Update, Delete) patient records
  • User-based dashboard interface
  • Notification feature for system requests
  • Status feature for recovered patients
  • Room management & availability updates

5. Development Phase Using Laravel

Backend Development
  • Data processed using Laravel Controllers
  • Authentication implemented using Laravel Auth
  • Middleware applied to separate user access
  • Validation on form submission
Frontend Development
  • Blade templating system used
  • Bootstrap and custom CSS styling for designs
  • Responsive layout for hospital data listings
